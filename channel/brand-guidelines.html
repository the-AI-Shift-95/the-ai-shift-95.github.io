Brand & UI Guidelines

Single source of truth for all demos, landing pages, and production interfaces. Every screen — job post, demo interface, lead capture form — uses this system. No exceptions, no dark themes.

Design Philosophy
💡
Light, not dark. The brand lives on a lavender-white background (#f8f7ff) with white card surfaces. Purple is an accent, not a background. This is a deliberate choice — it signals trust and clarity, not a hacker tool.
💡
System font only. No Google Fonts, no custom fonts. -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif. Fast, native, never looks wrong.
💡
Labels are loud, body is quiet. Section labels are uppercase, weight 800, tight tracking. Body copy is muted grey, not black. The contrast between them creates visual rhythm.
✅
White cards on lavender. Content lives in #fff cards on a #f8f7ff page. Cards have a #e9d5ff border — subtle purple, not grey.
✅
Purple is the only accent. Buttons, labels, highlights, borders — all purple. Never blue, never teal, never orange.
🚫
No dark backgrounds. Demo interfaces, dashboards, and loading screens all use the same light system. The branded wrapper is NOT dark.

Demo Interface Patterns

How to apply the brand system to demo screens. The demo interface uses the same light system — not a dark dashboard. The only difference is the addition of a top bar and output panels.

✅
Page background stays #f8f7ff. Same as every other page. The demo is not a special dark-mode exception.
✅
Output panels use #f8f7ff bg with #e9d5ff border. Same as the opp-box card — viewer immediately recognises the brand.
✅
Highlighted lines use the tint (#f5f3ff) with a #ddd6fe border. Same as checkbox hover — already in the system.
🚫
No dark backgrounds in demos. The viewer should never think "oh, that's some developer tool." It looks like the same clean interface they saw on the landing page.
🚫
No custom fonts, no gradients, no glow effects. The brand is flat and clean. Shadows only for card elevation (subtle).
💡
Loading states use the green dot + status label pattern. "Calendar connected" = green dot + grey text. Same as the job post meta line.
